## 📌 About the Internship

This internship focuses on building and understanding **agentic AI systems** using **AgenticGoKit** — a framework for creating intelligent, autonomous agents powered by LLMs.

Throughout this journey, I will explore:

* Agent-based architectures
* LLM integrations (OpenAI / Ollama)
* Streaming responses
* Observability & tracing
* Real-world AI applications
